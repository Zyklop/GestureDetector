Please consider the following steps before starting the program:
- KinectSAFramework.dll is present in the exe's directory
- Microsoft.Kinect.dll is present in the exe's directory
- In the image directory there must be present at least one valid 
  JPEG image.
- The Kinect device driver must be installed
- The Kinect device has to be connected and powered

Have fun,
