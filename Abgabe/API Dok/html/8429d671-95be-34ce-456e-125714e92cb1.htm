  Class Description
Condition
Base class for static gesture parts. They just succeed or fail.
DynamicCondition
Continuous gestures succeed after a certain time or on a certain condition. Before that, they just trigger that they occured. Gesture Sequence: -------|----------|----------|-----|------|--------|-----> T T T S T T T = fireTriggered S = fireSuccess An example is the zoom gesture. We always need feedback for the zooming factor, but we cant succeed. Therefore we trigger an occurence. The zoom gesture succeeds when zooming ended.
